Estrutura básica de um HTML 5

<header>texto</header>
Cabeçalho. Indica a introdução de um tema abordado na página. Quanto mais no começo do código está, mais relevante para os mecanismos de busca.

<nav>texto</nav>
Agrupa uma série de links, desde que sejam blocos de navegação principal (nav primaria, secundária e in page).

<section>texto</section>
Define uma seção dentro de um determinado elemento. Pode conter um header, um footer, e uma nova ordem de títulos, se necessário.
Pode ser utilizada na tag article para separar notícias, eventos, galeria de fotos, etc

<article>texto</article>
Trata-se de uma área que define o conteúdo principal da página, podendo ser uma composição de formulários, um artigo de notícias, etc. Desde que seja um conteúdo independente

<aside>texto</aside>
Faz referência ao conteúdo principal que o envolve (como se fosse separado do conteúdo principal)
quando está dentro de um <article>, faz referência ao assunto geral dele, se estiver fora, está relacionado com o conteúdo glogal do site. Pode ou não estar num sidebar

<footer>texto</footer>
Rodapé em um documento.
Pode ser utilizado dentro da tag section em um documento. Normalmente um <footer> contém as seguintes informações:
  • informações de autoria;
  • informações sobre direitos autoriais
  • informações de contato
  • mapa do site
  • voltar aos links principais
  • documentos relacionados
* informações de contato dentro de um elemento <footer> devem estar dentro de uma tag <address>.

<address>texto</address>
Define as informações de contato do autor / proprietário de um documento ou artigo.
As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc. O texto no<address>elemento geralmente é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois do elemento <address>.

Atributos HTML
Os atributos HTML fornecem informações adicionais sobre os elementos HTML (tags)
  1. Todos os elementos HTML podem ter atributos
  2. Os atributos são especificados na tag inicial
  3. Os atributos geralmente vêm em pares nome / valor como: atributo = "valor"
Atributos para o elemento <img>, que é a tag usada para embutir uma imagem em uma página HTML:
  • src: especifica o nome do arquivo (imagem) e respectivo caminho que pode ser uma URL absoluta ou relativa. (exemplo: src = "logo.png")
  • width e height: especifica a largura e a altura da imagem (em pixels). (exemplo: width="500" height="600")
  • alt: define um texto alternativo para a imagem quando por algum motivo não puder ser exibida. Isso pode ser devido a uma conexão lenta, um erro no src atributo ou se o usuário usa um leitor de tela. (exemplo: alt="Grafo de conveniência para o departamento de TI")
Atributos para diversos elementos:
  • style: é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais (exemplo: style = "color:blue;")
  • title: especifica um texto, estilo tooltip que aparece quando o usuário passa o mouse sobre o elemento. (exemplo: title="uma árvore não pode possuir caminho cíclico")
  • id: define um marcador dentro da página HTML. Os marcadores podem ser úteis se uma página da web for muito longa: funciona como ponteiros internos chamados por <a href = "#marcador"> para facilitar a navegação da página. (exemplo: id="w56QQ")
Nota:São muitas as possibilidades de utilização dos elementos HTML e a combinação entre eles. Aqui foram elencados algumas das tags indicadas no material de apoio da disciplina COM130, que requer um estudo muito mais profundo para uma utilização profissional e não acadêmica.
